加入特工

你是一名特工，你的的使命是进入曼哈顿区拯救在病毒大爆发之后遗留在那里的平民。和其他特工组队，一同恢复秩序，查找病毒的根源，并且夺回纽约。

  • 武器装备

  • 特殊技能

  • 行动基地

团结就是力量

在《全境封锁》的世界里你可以做一个独行侠。也可以组成四人小队击败最强大的敌人。使用对团队配合最有利的武器和技能。

x

武器装备

《全境封锁》是一款现代背景动作RPG：订制适合你的装备是成功的关键。击败敌人并收集装备会让你越来越强大。

x

特殊技能

随着游戏的进程你可以解锁特殊的技能。在战斗中你需要选择能应对当前形势的技能，聪明的使用技能会在战斗中占据优势。

x

行动基地

行动基地是你的主城，完成系列任务，就能够恢复基地内的医疗，技术和安全部门的运转，并且为自己提供便利和安全的休息区。

夺回纽约

《全境封锁》的故事发生在曼哈顿，病毒爆发的中心区域。这里是一片巨大的城市区域。你需要完成大量的任务来恢复秩序，并且查找病毒爆发的根源。

  • 残破的大都会

  • 探索地标场所

  • 世界任务

  • 一个真实存在的世界

残破的大都会

纽约市在病毒爆发中遭到了毁灭性打击，整个城市陷入混乱。市民们在恐惧中试图逃命，往日的生活已无法恢复。

x

探索地标场所

到世界的特殊地点探索，看看社会的崩坏带来的后果。麦迪逊花园广场，时代广场甚至纽约的地下设施都面目全非。

x

世界任务

《全境封锁》的世界充满了开放性的世界任务，有主线剧情也有随机遭遇。同时没后会有新的内容和挑战公布提供长期的游戏体验。

x

一个真实存在的世界

进入一个昼夜天气变化都会影响你体验的世界。利用这个世界的变化在战斗中获得优势，升级你的防毒面具进入被严重污染的区域。

暗区来袭

在无缝连接的巨大世界中，有一块隐藏在高墙之后的特殊区域。这里有最好的装备也是游戏中最危险的区域，恐惧，背叛，紧张是这里的主题。

  • 多人互动区域

  • 反叛特工

  • 装备输送

  • 高科技装备

多人互动区域

暗区是一个充满恐惧和背叛的地方。不但要面对敌人还要担心其他玩家的背叛，其他玩家可能打黑枪抢走你在暗区的背包。

x

反叛特工

在暗区中杀死其他玩家会变成反叛特工。在一段时间内你的位置会出现在所有人的地图上，并可能被所有人追杀。

x

装备输送

由于暗区内的病毒非常泛滥，意味着你在暗区得到的战利品都是被感染过的。你需要把这些战利品输送回基地进行净化之后才能使用。找到你附近的运输点，发射闪光弹并且呆在原地直到直升机到达，当心不要被其他人杀掉抢走你的暗区装备。

x

高科技装备

暗区是病毒爆发的源头，也是首先成为无人区的区域。军队在撤退时被迫留下了许多先进的武器和设备。即使这里充满危险，仍然吸引着前来冒险的特工们。
